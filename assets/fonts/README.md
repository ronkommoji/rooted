# Font Files

Place your Canela font files in this directory:

- `Canela-Regular.ttf` - Regular weight
- `Canela-Bold.ttf` - Bold weight  
- `Canela-Medium.ttf` - Medium weight (optional)

## Getting Canela Font

Canela is a commercial font. You can purchase it from:
- Commercial Type: https://commercialtype.com/catalog/canela
- Or your font provider

After downloading, place the font files in this directory and the app will automatically load them.
